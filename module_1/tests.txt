Тестовый сценарий 1.1.1 Отображение информации о добавленном в корзину товаре
Цель: проверить, что информация о добавлении товара в корзину и его стоимость отображаются как инфо-сообщения в каталоге.
Предусловие:
- 1. Пользователь не зарегистрирован на сайте.
Шаги для воспроизведения:
- Шаг 1. Перейти в раздел каталога "Все продукты"; отображается каталог продуктов, у каждого продукта имеется картинка, оценка, название, цена, наличие на складе, кнопка "Добавить в корзину".
- Шаг 2. Для одного из продуктов в списке на первой странице нажать кнопку "Добавить в корзину".
Результат:
1. Отображено инфо-сообщение о добавлении товара в корзину, с указанием названия.
2. Отображается инфо-сообщение с указанием стоимости товара.
3. Отображается инфо-сообщение об удовлетворении условиям Deffered benefit offer.
3. Отображается инфо-сообщение об удовлетворении условиям Shipping offer.

Тестовый сценарий 2.1.1 Переход к офрмлению заказа из корзины под гостем с применением скидки
Цель: проверить, что доступен переход к оформлению товаров из корзины с применением кода скидки.
Предусловие:
- 1. Пользователь не зарегистрирован на сайте.
- 2. Пользователь добавил в корзину 3 товара.
Шаги для воспроизведения:
- Шаг 1. Перейти на страницу "Корзина" путем нажатием кнопки "Посмотреть корзину"; загружена страница с выбранными товарами.
- Шаг 2. Нажать кнопку "У меня есть скидка"; отображено поле ввода кода скидки с кнопкой подтверждения.
- Шаг 3. В поле ввода ввести код скидки, сохранить; ввод совершен, скидка применена, цена в корзине изменилась с учетом скидки.
- Шаг 4. Нажать кнопку "Оформить заказ".
Результат: соврешен переход на страницу персонализации.

Тестовый сценарий 2.2.1 Полное удаление товаров из корзины
Цель: проверить, что доступно полное удаление товаров из козины.
Предусловие:
- 1. Пользователь не зарегистрирован на сайте.
- 2. Пользователь добавил в корзину 3 товара.
Шаги для воспроизведения:
- Шаг 1. Перейти в на страницу "Корзина"; загружена страница с выбранными товарами.
- Шаг 2. Нажать кнопку удаления 1-го в списке товара; товар удален.
- Шаг 3. Нажать кнопку удаления 2-го в списке товара; товар удален.
- Шаг 4. Нажать кнопку удаления 3-го в списке товара; товар удален.
Результат: коризна пуста, отображается кнопка "Продолжить покупки".

Результат: совершен переход на страницу ввода данных адреса доставки.
